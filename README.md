# üåø Wildlife Realm - 3D Predator Challenge

#### Description:

**Wildlife Realm** is an exhilarating **3D game** where players step into the paws of the planet's two fastest land animals. Built with **Next.js** for a modern web experience and leveraging **Babylon.js** for immersive 3D graphics, this project challenges players to master the speed, agility, and hunting skills of these apex predators in a dynamic, open environment.

---

## üåü The Vision

Have you ever wondered what it's truly like to run at top speed? This project brings that incredible experience to life. Players get to choose between the **cheetah** and the **peregrine falcon** (as the two fastest animals on earth, though one flies!), offering unique gameplay centered on extreme velocity. Our goal is to create a realistic, high-speed simulation that is both thrilling and respectful of these magnificent creatures' natural abilities.

---

## ‚ú® Key Features

- **Dual-Animal Gameplay:** Choose between the **Cheetah** (fastest land animal) and the **Peregrine Falcon** (fastest animal overall) for distinct playstyles.
- **Immersive 3D World:** Explore a beautifully rendered 3D environment optimized for high-speed action and built with the power of Babylon.js.
- **Speed & Agility Mechanics:** Experience realistic mechanics for acceleration, top speed, and sharp turning in both animals.
- **Modern Tech Stack:** A fast and responsive experience powered by Next.js and styled with a clean **Tailwind CSS** interface.
